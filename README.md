📚Library Management System

A web-based Library Management System built using **Node.js, Express, HTML, CSS, and JavaScript**.  
This project allows users to browse books, manage borrow/return records, and handle user authentication.

🚀 Features
- 📖 "Book Catalog"– View available books by category (CSE, ECE, EEE, IT, etc.).
- 🔍 "Search Functionality" – Search books by title or author.
- 👤 "User Authentication" – Login and signup pages for students.
- 📅 "Booking Slots" – Reserve and borrow books online.
- 📜 "Borrow/Return Records" – Track borrowed books and due dates.
- 📞 "Contact Page" – Simple contact form for queries.

🛠️ Tech Stack
- "Backend:" Node.js, Express.js  
- "Frontend:" HTML, CSS, JavaScript  
- "Database:" (Optional) Can be integrated with MySQL/MongoDB  
- "Other:" `.env` for environment variables

⚙️ Installation & Setup

1. "Clone the repository"
   ```bash
   git clone https://github.com/JahnaviNekkalapu/LibraryManagement.git
   cd LibraryManagement
   
2. Install Dependencies
     npm install
   
3. Configure Environment Variables
   - Create a .env file in the root directory.
    - Add values like:
      PORT=3000
      DB_URL=your_database_url_here

4.Run the server
  node server.js
